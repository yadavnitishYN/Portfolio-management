Request a fresh statement from CAMS

This app runs in a browser sandbox, so it genuinely cannot log in to camsonline.com and press buttons there for you — web pages are blocked from controlling other websites, and CAMS additionally uses human-verification checks. Two real ways to automate it: use Claude in Chrome (Anthropic's browser agent, which can fill this form under your supervision), or ask Claude in chat for a small Playwright script you run on your own computer. From inside this app, the fastest path is the one-click checklist below — CAMS emails the statement within minutes.
  1. Statement Type → select Summary
  2. Folio Listing → select Transacted Folios and Folios with Balance
  3. Email → the email registered against your mutual fund folios
  4. Password & Confirm Password → choose any password; remember it, it unlocks the PDF
  5. Click Submit — the PDF arrives in your inbox within a few minutes
  6. Come back here → Update Mutual Funds tab → upload the PDF with that password

AI provider & API key

Automatic reading of statements and FD/bond documents needs an API key from either OpenAI (platform.openai.com → API keys) or Anthropic (console.anthropic.com). Pick your provider, paste the key once — it's saved only in this browser on this device, never written into the app's code, and sent only to that provider's API. Each document read costs a fraction of a rupee.

Backup & restore

Your snapshots, liquid assets, and archived statement PDFs live in this browser only — clearing browser data or switching devices loses them. Export a backup file now and then, and import it on any device to restore everything. Your API key is not included in backups.
